项目名称： 获得页面html
知识范围： chrome控件 js html
版本号  ： 1.2

程序说明：获得页面html
功能说明：打开页面，点击控件图标，获得页面信息
辅助说明：
    （默认） 1、创建google控件（√）
    （用户） 2、点击控件后获得页面信息（√）

文件说明：
    manifest.json google控件配置文件
    background.*  后台运行程序
    content.js    插入到页面中的脚本
    popup.*       控件展示界面


使用说明：
    1、打开google浏览器的扩展程序
    2、加载已解压的扩展程序
    3、选择文件即可在扩展程序区域看到我们的控件
    4、打开https://www.baidu.com/页面，点击右上角的图标，点击操作，即可看到页面body的内容

其他说明：适合google浏览器
