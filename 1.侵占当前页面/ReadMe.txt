项目名称： 侵占当前页面
知识范围： chrome控件 js html
版本号  ： 1.1

程序说明：侵占当前页面
功能说明：打开页面，点击控件图标，改变页面信息
辅助说明：
    （默认） 1、创建google控件（√）
    （用户） 2、点击控件后改变页面信息（√）

文件说明：
    manifest.json google控件配置文件
    backgroung.* 后台运行程序
    popup.*      控件展示界面
    func.js      插入到页面中的脚本
    jquery-3.3.1.min.js jquery脚本


使用说明：
    1、打开google浏览器的扩展程序
    2、加载已解压的扩展程序
    3、选择文件即可在扩展程序区域看到我们的控件
    4、打开任意页面，点击右上角的图标，点击操作，即可看到 你的电脑被我占领了

其他说明：适合google浏览器
